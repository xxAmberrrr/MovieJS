# MovieJS
Opdracht voor Serious JavaScript

De opdracht is uiteindelijk iets anders geworden. Eerste wilde ik met de omdb API gaan werken alleen ik kwam er achter dat je maar 1 film per keer op kunt halen en dus niet een hele collectie aan films. 
Omdat dit dus een probleem was heb ik nu mijn eigen API gebruikt en die wat aangepast zodat ik ermee kon werken voor deze opdracht en zodat ik genoeg collecties had.

Uiteindelijk kan de opdracht dit:
- Wanneer je op de movies button klikt krijg je alle movie data op het scherm te zien die in mijn API aanwezig is
- Wanneer je op 1 bepaalde film button klikt krijg je alleen de data van die specifieke film te zien.

Hetzelfde geld voor genres en actors.
